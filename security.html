🔒 Ethical Framework

Privacy & Ethical Framework

AERIS is designed with privacy and ethical considerations as foundational requirements, not afterthoughts.

Data Protection

Data Protection Approach

Core privacy mechanisms designed into the AERIS system architecture.

🔐

Role-Based Access Control

Each user role (student, teacher, counselor, admin) has strictly defined data access boundaries. No role can access data outside its scope.

🔗

Encryption in Transit

All data transmitted between client and server is encrypted using standard transport layer security protocols.

🏫

Institution-Controlled Data

All student data remains under the control of the deploying institution. Data governance policies are institutionally defined.

🔬

Federated Learning Direction

Research direction exploring federated learning to enable model improvement without centralizing institutional data.

Compliance Strategy

Aligned with Indian DPDP Act Principles

The AERIS privacy framework is designed to align with the principles outlined in India's Digital Personal Data Protection Act.

→
Lawful Purpose

Data processing limited to legitimate educational wellbeing purposes.

→
Data Minimization

Only data essential for emotional risk assessment is collected.

→
Purpose Limitation

Student data is not used for purposes beyond emotional wellbeing support.

→
Consent Framework

Institutional and guardian consent mechanisms built into the system.

🛡️

Ethical AI Principle

AERIS is designed to assist institutional decision-making, not replace it. All critical assessments are intended for human review.
