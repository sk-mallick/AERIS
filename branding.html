Visual Language

Interface Components

Core design patterns used throughout the AERIS interface.

Glassmorphism Cards

Frosted glass effect for information panels in dark UI sections. Provides depth and visual hierarchy.

Data Visualization

Semantic color scale (Teal → Sky → Amber → Coral) used consistently for risk-level representation.

Scroll Animations

Subtle reveal animations on scroll for progressive content disclosure. Keeps interface calm and focused.
